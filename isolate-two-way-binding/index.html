scope : {title: @msdTitle, subtitle: '=msdSubtitle'}

  • controller and directive have separate scopes.
  • Directive title is set to read-only access of controller's title.
  • If 'Bob it!' is clicked, connection between to title values is severed.
  • If 'Subtitle it!' is clicked, parent controller subtitle is changed because of two-way binding (=) with subtitle
  • Biggest difference between this and scope: false -- we're explicity specifying the properties we bind
